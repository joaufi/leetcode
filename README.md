# leetcode
Answers  to leetcode.com questions

# to run

```
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
pytest <question_folder_name>/tests.py
```

Example:

```
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
pytest lc42_trapping_rain_water/tests.py
```
